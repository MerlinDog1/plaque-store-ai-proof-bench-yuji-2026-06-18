# Milestones

## M0 - Repo And Plan

Deliverables:

- standalone git repo;
- `SPEC.md`;
- `MILESTONES.md`;
- `STATUS.md`;
- initial commit pushed to GitHub.

Acceptance:

- `git status --short --branch` is clean on `main`;
- GitHub repo exists and contains the plan files.

## M1 - Runnable Static Storefront

Deliverables:

- minimal app shell;
- homepage with hook;
- product category cards;
- local dev server.

Acceptance:

- app returns HTTP 200 locally;
- screenshot captured.

## M2 - Product Builder

Deliverables:

- category/size/material/mounting controls;
- wording input;
- price summary.

Acceptance:

- changing controls updates price and summary.

## M3 - Instant Text Proof

Deliverables:

- SVG plaque preview;
- deterministic text fitting;
- material styling;
- border/fixing preview.

Acceptance:

- sample long and short wording both render inside the plaque.

## M4 - Basket And Placeholder Checkout

Deliverables:

- basket summary;
- proof/order details saved locally;
- checkout placeholder screen.

Acceptance:

- configured item can be added to basket and viewed in placeholder checkout.

## M5 - Polish And Deployment

Deliverables:

- responsive pass;
- final screenshot set;
- live tunnel or deploy preview.

Acceptance:

- desktop and mobile screenshots reviewed;
- public URL returns HTTP 200.
